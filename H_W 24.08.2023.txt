int max = array[0];
for (int i = 1; i < array.length; i++){
	if (array[i] > max) {
		max = array[i];
	}
}
Алгоритм является линейным, поскольку реализует поиск наибольшего по значению элемента неотсортированного массива проходясь по каждому элементу только 1 раз(поскольку в теле цикла нет вложенных сложных конструкций не считая операции сравнения) и в общей сложности количество операций равно array.length .
Поэтому оценку сложности алгоритма можно присвоить О(n).
